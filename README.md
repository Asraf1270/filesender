# OneTimeSend

## Setup
1. Place in `htdocs/onetime-send`
2. Update `BASE_URL` in `config.php`
3. Set `.env` password
4. Access: `http://localhost/onetime-send/public`

## Features
- One-time download
- QR code
- Admin panel
- Auto-expire (24h)
# OneTimeSend (No Local QR)

- No PHP QR library
- QR via https://api.qrserver.com (free)
- Works on XAMPP, Apache, PHP built-in server
- One-time download
- Admin panel
